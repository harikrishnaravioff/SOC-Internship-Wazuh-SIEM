# SOC Internship Project — Wazuh SIEM Lab

A 4-week hands-on SOC internship project covering SIEM deployment, log monitoring, attack simulation, and professional dashboarding using Wazuh on Oracle VirtualBox.

---

## About

This project documents a complete Security Operations Center (SOC) internship simulation built around the Wazuh open-source SIEM platform. Each week focused on a different phase of real-world SOC work, from initial lab setup to threat detection and professional reporting.

---

## Tools & Environment

- Wazuh 4.14.5 (Manager + Agent)
- Oracle VirtualBox
- Kali Linux 2026.2
- OpenSearch / Kibana Dashboard
- MITRE ATT&CK Framework

---

## Week Breakdown

| Week | Topic | Key Activity |
|------|-------|--------------|
| Week 1 | Lab Deployment | Wazuh Manager + Kali Agent setup on VirtualBox |
| Week 2 | Log Monitoring & Threat Hunting | Authentication event analysis, MITRE mapping |
| Week 3 | Attack Simulation & Detection | Brute force, EICAR file, command monitoring |
| Week 4 | Dashboard & Final Report | Kibana visualizations, SOC-style reporting |
